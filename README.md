# Phần mềm Quản lý Bán vé Minera Hot Springs Bình Châu

## 1. Giới thiệu dự án
Hệ thống quản lý quy trình bán vé, kiểm soát vé và thống kê doanh thu cho khu phức hợp Minera Hot Springs. 

## 2. Công nghệ sử dụng
*   **Backend:** C#, ASP.NET MVC
*   **Database:** SQL Server
*   **Frontend:** HTML, CSS, JavaScript (hoặc các thư viện hỗ trợ)

## 4. Hướng dẫn cài đặt (Dành cho thành viên tải code về máy)
1. Clone dự án về máy: `git clone [https://github.com/ThanhTu-Le/quan-ly-ban-ve-minera]`
2. Mở solution bằng Visual Studio.
3. Chạy file script SQL (nằm trong thư mục Database) để khởi tạo CSDL SQL Server.
4. Cập nhật chuỗi kết nối (Connection String) trong file `Web.config`.
5. Build và chạy dự án (F5).
